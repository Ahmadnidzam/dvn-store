# DVNStore — Working Rules

## Bahasa & Komunikasi
- Selalu jawab dalam Bahasa Indonesia (kecuali user menulis dalam bahasa lain).
- Selalu MINTA IZIN sebelum mengubah file. Tunjukkan checklist file + diff garis besar.
- Sebelum tugas multi-step: gunakan AskUserQuestion untuk klarifikasi.
- Setiap tugas non-trivial: buat TodoList dengan langkah verifikasi di akhir.
- Setelah selesai: ringkas perubahan, jangan ulang isi file.

## Arsitektur Wajib
- Role-based: `user`, `developer`, `admin` (admin via seeder, tidak via form).
- Controllers terorganisir per role: `app/Http/Controllers/{User,Developer,Admin,Payment}/`.
- Middleware `role:user|developer|admin` di semua route.
- Tidak ada lagi kategori `film` — hanya `app` dan `game`.
- Wallet pakai pattern `wallets` + `wallet_transactions` (audit-able).
- Database baru di `dvn_store_v2` (jangan kembali ke `dvn_store` lama).

## Aturan Keuangan
- Upload fee: Rp 10.000 untuk setiap upload (config: `DVN_UPLOAD_FEE`).
- Platform fee: 10% dari setiap penjualan (config: `DVN_PLATFORM_FEE_PERCENT`).
- Minimum withdraw: Rp 50.000 (config: `DVN_MIN_WITHDRAW`).
- Semua nilai uang disimpan sebagai integer Rupiah, bukan float.
- Pembayaran via Midtrans Snap; withdraw via Midtrans IRIS Disbursement.

## Keamanan Upload
- Semua file installer di-scan VirusTotal lewat queue (`ScanUploadedFileJob`).
- Platform baru `is_published=false` sampai `scan_status='clean'`.
- File `infected` dihapus otomatis, fee tidak refund.

## Pola Kode
- Wallet: SELALU pakai `credit()`/`debit()`, jangan update `saldo` langsung.
- Semua webhook handler harus idempotent (cek status sebelum proses ulang).
- Pakai `DB::transaction` + `lockForUpdate()` untuk semua mutasi keuangan.
- Try-catch panggilan eksternal (Midtrans, IRIS, VirusTotal); rollback transaksi draft kalau gagal.
- Genre & kategori ambil dari `config/dvnstore.php`, validasi dengan `Rule::in()`.

## Midtrans
- WAJIB: `MIDTRANS_IS_PRODUCTION` flag harus cocok dengan prefix key, kalau mismatch → 401.
- IRIS API key berbeda dari Snap server key — jangan dicampur.
- Default development: SELALU sandbox. Jangan asumsikan production sudah aktif.

## File yang TIDAK Boleh Diubah Tanpa Izin Eksplisit
- `bootstrap/app.php` (kecuali tambah middleware alias).
- `composer.lock` (regenerated by composer).
- File di `vendor/`, `storage/framework/`, `bootstrap/cache/`.
- `database/migrations/0001_01_01_*` (default Laravel — biarkan saja).

## UI/UX
- User menggunakan skill UI/UX khusus di Claude Code CLI.
- Jangan redesign UI tanpa izin. Fokus ke logika.
- Layout saat ini sederhana Bootstrap 5.3 — biarkan user yang upgrade.

## Style Bahasa Indonesia
- Pakai istilah teknis Inggris untuk programming (controller, route, migration).
- Pakai istilah Indonesia untuk domain (pengguna, transaksi, pembayaran).
- Jangan terlalu formal — tetap natural seperti percakapan.

## Aturan penghematan token
- Tidak perlu membaca semua directory untuk membaca kode saat pembuatan laporan kecuali sudah konfirmasi kembali disesi diskusi
- saat pembuatan laporan kamu hanya akan mendapat akses membaca struktur file dari struktur_file_prompt.md saja
- untuk pembuatan laporan, pastikan sejalan dengan laporan_context_prompt.md
